# Octoperf Jenkins Plugin

Run load tests from Jenkins CI with [OctoPerf](https://octoperf.com). For more information, see our documentation on [Continuous Load testing with Jenkins CI](https://api.octoperf.com/doc/runtime/edit-scenario/integrations-automation/ci-cd/ci/jenkins/).

## Compatibility

Jenkins plugin `5.0.0` and higher is only compatible with OctoPerf `16.0.0` and higher.
